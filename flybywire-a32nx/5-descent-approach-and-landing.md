# Descent, Approach and Landing

## Descent and Approach Planning

- Calculate the required distance for descent (flyPad / Performance / Top of Descent)
- [ATIS Page](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/mcdu/mcdu-menu/#atis-page): MCDU MENU / ATSU / AOC MENU / ATIS - select SEND
- [AOC MENU Page](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/mcdu/mcdu-menu/#aoc-menu-page): MCDU MENU / ATSU / AOC MENU  - select RECEIVED MESSAGES / PRINT
- [Performance Page](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/mcdu/perf/): MCDU / PERF - fill APPROACH

## Starting the Descent

### At TOD (top of descent)

- Request descent clearance
- [Altitude selector](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/glareshield/fcu/#altitude-selector-knob-inner-and-outer): ALT selector - set
- [Altitude selector](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/glareshield/fcu/#altitude-selector-knob-inner-and-outer): ALT selector - push to set Managed Altitude Mode

## Intercepting ILS

### Within the IAP (Instrument Approach)

- [Performance Page](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/mcdu/perf/): MCDU / PERF - activate APPR phase
- [Speed button](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/glareshield/fcu/#spdmach-button): SPEED knob - push to set Managed Speed Mode
- [Flaps](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/flaps/): FLAPS lever - set 1 when V_FE -15 knots
- [Approach Mode](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/glareshield/fcu/#appr): APPR pushbutton - press

### When FMA / LOC in green

- [Flaps](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/flaps/): FLAPS lever - set 2 when S-speed

### When FMA / G/S in green

- [Gear](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/front/autobrake-gear/): LANDING GEAR lever - set DOWN at between 2500ft and 1500ft
- Lights:
    - [Runway Turnoff Lights](https://docs.flybywiresim.com/pilots-corner/a380x/a380x-briefing/flight-deck/ovhd/ext-lt/#rwy-turn-off): RWY TURN OFF switch - set ON
    - [Nose Lights](https://docs.flybywiresim.com/pilots-corner/a380x/a380x-briefing/flight-deck/ovhd/ext-lt/#nose): NOSE switch - set T.O
- [Flaps](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/flaps/): FLAPS lever - set 3
- [Flaps](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/flaps/): FLAPS lever - set FULL
- [Autobrake](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/front/autobrake-gear/#autobrk): AUTO/BRK - set as required (LO when dry, MED when wet or short runway)
- [Speed Brake](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/speedbrake/): SPEED BRAKE lever - set ARM
- [Call All Cabin Crew](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/ovhd/calls/#all): CALLS ALL pushbutton - press

> [!NOTE] flyPad
> Perform LANDING checklist

## Landing

- [Thrust Lever](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/thrust-pitch-trim/): THRUST lever - set REV FULL on touchdown of the main gear
- [Thrust Lever](https://docs.flybywiresim.com/pilots-corner/a32nx/a32nx-briefing/flight-deck/pedestal/thrust-pitch-trim/): THRUST lever - set IDLE at about 60 knots
- Brake manually at about 40 knots

## Vacate Runway

- Stop on the taxi way
